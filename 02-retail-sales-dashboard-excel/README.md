# Interactive Sales Dashboard — Retail Performance Analysis

**Tools:** Microsoft Excel | Pivot Tables | Interactive Slicers | Data Visualization

## Overview
An interactive Excel dashboard analyzing retail sales performance across two stores, multiple product categories, sales managers, and UK locations.

## Dataset Scope
- £18.3M in total retail sales analyzed
- 2 stores: Computech and MicroWorld
- 9 product categories
- 10 sales managers
- 7 months of trading activity across multiple UK locations

## What Was Done
- Built dynamic pivot tables summarizing performance by store, product category, manager, and month — enabling multi-dimensional analysis from a single dataset.
- Designed interactive slicers allowing non-technical stakeholders to self-serve insights by filtering by store, category, and time period.

## Key Insights
- Seasonal dip in sales observed in June.
- Laptops and PCs were the top-performing product categories.
- A 2x revenue gap was identified between the highest and lowest performing sales managers.

## Key Files
- `sales_dashboard.xlsx` — full interactive dashboard
- `visuals/` — screenshots of dashboard views

## How to Use
Open `sales_dashboard.xlsx` in Microsoft Excel, then use the slicers to filter by store, category, and time period.
