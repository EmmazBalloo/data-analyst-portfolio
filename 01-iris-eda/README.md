# Iris Dataset — Exploratory Data Analysis (EDA)

**Tools:** Python | pandas | NumPy | Matplotlib | Seaborn

## Overview
A comprehensive exploratory data analysis of the classic Iris dataset, focused on understanding the statistical properties and distributions of each feature across the three iris species.

## What Was Done
- Performed descriptive statistics (mean, median, mode) for all numerical features.
- Conducted spread analysis using standard deviation and variance.
- Generated histograms and box plots to identify species-level differences across sepal and petal measurements.
- Applied percentile analysis (Q1, Q3) to lay the groundwork for IQR-based outlier detection.
- Documented all findings in a clean, well-structured notebook suitable for portfolio presentation.

## Key Files
- `iris_eda.ipynb` — main analysis notebook
- `visuals/` — exported charts and plots

## How to Run
```bash
pip install pandas numpy matplotlib seaborn jupyter
jupyter notebook iris_eda.ipynb
```
